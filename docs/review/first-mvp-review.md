# First MVP Skeleton Review

## Active Artifact

- Artifact id: `fff-mvp-skeleton-001`
- Title: Fast Fiction Factory MVP Skeleton Review Workbench
- Path: `public/review/index.html`
- Open command: `Invoke-Item .\public\review\index.html`

## Review Purpose

Confirm that a raw memo can be transformed into reviewable structured candidates without external publishing, credentials, paid services, destructive migrations, automated YouTube upload, or final canon decisions.

## Review Checklist

- Raw memo intake is visible and editable.
- Generated output includes all required extraction categories.
- Profile, claim, timeline, task card, and outline candidates are visible.
- Every candidate has `adopt`, `provisional`, `hold`, and `reject` controls.
- QA panel covers all required gates.
- Decision log records local review choices.
- Artifact manifest identifies the review output and validation command.

## Current Scope

This is a local-first skeleton. It proves the review surface and data contract before real extraction, persistence, video generation, or publishing adapters are considered.
