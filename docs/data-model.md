# Fast Fiction Factory Data Model

All entities are local-first MVP records. The prototype uses mocked data in `public/review/index.html`; this document defines the intended shape for later persistence.

## Shared Types

```ts
type ReviewStatus = "adopt" | "provisional" | "hold" | "reject";
type SourceTrust = "author_memo" | "derived_candidate" | "external_unverified" | "rights_unknown";
type GateStatus = "pass" | "warn" | "block" | "not_checked";
```

## Portable Project State Envelope

The JSON export/import and local file persistence slices use a versioned envelope. Unknown imported top-level fields are not applied to the UI, but they are preserved under `metadata.importedUnknownFields` on the next export.

`artifacts/sample-project-state.json` is the stable review sample. `artifacts/current-project-state.json` is the normalized local working copy generated by `tools/fff-state.mjs`; it remains a repo-local review artifact, not database persistence.

```ts
type ProjectStateEnvelope = {
  schemaVersion: "fff.projectState.v1";
  exportedAt: string;
  project: Project;
  work: Work;
  rawMemo: string;
  extractedCandidates: {
    elements: Array<{ category: string; items: unknown[] }>;
    profiles: Profile[];
    claims: Claim[];
    timelineCandidates: TimelineEvent[];
  };
  reviewStatuses: Record<string, ReviewStatus>;
  reviewStatusBuckets: {
    acceptedCanon: Array<{ id: string; type: string; title: string; reviewStatus: "adopt" }>;
    provisionalCandidate: Array<{ id: string; type: string; title: string; reviewStatus: "provisional" }>;
    heldCandidate: Array<{ id: string; type: string; title: string; reviewStatus: "hold" }>;
    rejectedCandidate: Array<{ id: string; type: string; title: string; reviewStatus: "reject" }>;
  };
  unresolvedCreativeDecisions: UnresolvedCreativeDecision[];
  taskCards: TaskCard[];
  qaGateResults: QualityGateResult[];
  generatedOutlines: OutputPackage[];
  decisionLog: DecisionLog[];
  metadata: {
    viewerFacingInformation: string;
    productionOnlyNotes: string[];
    importedUnknownFields: Record<string, unknown>;
    activeArtifactId?: string;
    profileGhostFlow?: {
      requiredProfileTypes: Profile["profileType"][];
      requiredGhostNodeStatuses: Profile["ghostNodeStatus"][];
      humanOwnedDependencies: string[];
    };
  };
};
```
