Completed features

  • Raw memo intake and deterministic mock extraction.
  • Candidate review statuses: adopt, provisional, hold, reject.
  • Task cards with timing, utility, and risk fields.
  • 1-minute, 3-minute, 10-minute, and series outline candidates.
  • Portable JSON export/import with schema validation.
  • Explicit local file save/load with browser fallback.
  • Status filters, search, and collapse/expand controls.
  • Claim Ledger grouping by scope, truth, reality, risk, dependency, review status, and source.

Pending features

  • Timeline view.
  • Profile page and ghost node flow.
  • Real extraction engine and durable project database.

Not implemented

  • No YouTube publishing or upload adapter.
  • No AI video generation.
  • No database persistence.
  • No remote sync or production sync.
  • No final canon decision for Toma, the brass moth, or the Council.
